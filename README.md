# Cybersecurity Threat Monitoring & Incident Response Lab

This project demonstrates hands-on cybersecurity monitoring, threat detection, and incident response in a simulated environment, reflecting junior SOC analyst responsibilities.

## Project Overview
A virtual lab environment was set up to simulate real-world security events including brute-force attacks, port scans, suspicious outbound traffic, and log analysis.  

The objective was to monitor traffic, detect threats, document incidents, and perform remediation steps.

## Lab Environment
- Windows 10 Virtual Machine (victim machine)  
- Kali Linux Virtual Machine (attacker machine)  
- Wireshark for traffic capture and analysis  
- Windows Event Viewer for log monitoring  

## Security Incidents Simulated
1. Brute-force login attempts on Windows RDP  
2. Port scanning detection  
3. Suspicious outbound traffic monitoring  
4. Benign malware behaviour review  
5. Event log anomaly detection  

## Tools Used
- Wireshark  
- Windows Event Viewer  
- VirtualBox / VMware  
- Bash / PowerShell for testing  
- Network simulation scripts  

## Skills Demonstrated
- Packet capture and traffic analysis  
- Threat detection and monitoring  
- Incident documentation and reporting  
- Security log analysis  
- Basic incident response workflow  

## Outcome
Successfully identified and documented multiple simulated security incidents, demonstrating readiness for entry-level SOC or cybersecurity roles.
